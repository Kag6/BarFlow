# BarFlow

BarFlow is a lightweight sales and inventory tracking tool designed for small and medium-sized businesses that do not yet have a dedicated ERP, stock, or sales tracking system.

The project focuses on fast setup, simple data entry, and practical business visibility using familiar file-based data sources such as Excel or JSON.

## Purpose

Many small businesses manage sales, stock, and daily operations manually. BarFlow aims to provide a simple bridge between manual records and structured decision-making.

## Planned / Current Capabilities

- Sales and stock tracking from Excel or JSON data
- Simple product and transaction records
- Basic reporting for stock movement and sales performance
- No database requirement for the initial version
- Portable single-user workflow

## Target Users

- Small retailers
- Local service businesses
- Inventory-light operations
- Businesses that need a quick operational dashboard before investing in a full ERP system

## Tech Direction

- File-based data input: Excel / JSON
- Lightweight local-first application structure
- Future dashboard and reporting layer
